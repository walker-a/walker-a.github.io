PORTFOLIO

Media Tracker

Media Tracker is a mobile application I've been developing in my free time that allows users to organize their thoughts on the movies, books, games, and TV series they consume. Users can then follow each other to see what their friends are watching and what their thoughts are. It's built using react native with a firebase backend.

  • Project Repo: https://expo.io/@skywalker13/MediaTracker
  • Date: May 2020

Genetic Art

Genetic art is a project that I built with Julia Connelly for our Evolutionary Computing class at Carleton. It expands on a code base provided for the class and uses an evolutionary algorithm to evolve images toward (hopefully) art.

  • Project Repo: https://github.com/walker-a/GeneticArt
  • Date: May 2017

Ray Tracing

This project is a ray tracing graphics engine that Jack Wines and I built for our final project in Carleton's graphics class. It uses ray tracing to emulate reflections, shadows, and point light sources in a given scene (in this case, 4 spheres and walls).

  • Project Repo: https://github.com/walker-a/RayTracing
  • Date: Fall 2017
